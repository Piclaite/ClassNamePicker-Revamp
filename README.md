ClassNamePicker-Revamp基于ClassNamePicker更改而成，至少我自己用的还算顺手<br/>

### 主要改动
- [x] 主程序界面简化
- [x] 可用的语音播报
- [x] 开机自启动功能
- [x] 悬浮窗自定义图片
- [x] 悬浮窗基础样式变为灰色半透明圆角矩形，文字自适应悬浮窗尺寸
- [x] 悬浮窗可以选择显示单或双悬浮窗和是否自动贴边，贴边时若显示文字会自动改为‘点名’二字
- [x] 在选择重复抽取时可以设置最小抽取间隔
- [x] 抽男/女的选项在未勾选“性别选项”时隐藏
- [x] 程序运行

### 基础抽取

- [x] 从整个班级的名单中随机抽取一人
- [x] 选择从班级的男生/女生名单中随机抽取一人
- [ ] 均衡抽取男生/女生
- [x] 以上抽男/女在未勾选“性别选项”时隐藏
- [x] 可选是否允许重复抽到一人
- [x] 在选择重复抽取时可以设置最小抽取间隔 
- [x] 可保存已抽取过人的名字，并在下次打开后读取

### 互动玩法

- [x] 最小化后显示悬浮窗，可快速从课件切换
- [x] 悬浮窗可以使用自定义图片，选择单双悬浮窗和是否自动贴边
- [x] 抽取一人后自动开启计时器，适用于限时背诵等场景
- [x] 语音播报抽到人的名字
- [x] 统计已抽人数、剩余人数、概率、总抽取次数等信息
- [x] 简易的抽取动画，可在配置文件中自定义时长

### 名单编辑

- [x] 读取 TXT 文件，每行一个名字，支持从表格中直接复制名字一列
- [x] 只需再编辑女生名字，即可自动算出男生名字列表
- [x] 可以自动检测女生名单中的额外姓名并有一定的自动修改能力

## 开始使用
### 首次安装

只适配Windows系统。

下载完成后，将软件压缩包解压，解压时请不要解压到C盘根目录、【下载】文件夹中，否则可能会出现**文件无法读写、文件丢失**等问题

先启动一次程序，然后进入'_internal'文件夹中的`PickNameConfig`文件夹，编辑`names.txt`，每行填写一个名字（可从含有学生名字的表格中直接复制名字一列），然后可选择编辑`g_names.txt`，即为女生名单，男生名单将自动算出

如您有 Python 环境，可以直接拉取项目，（自行安装依赖）并运行`ClassNamePicker.py`。
